# 🛍️ Shopping Whisperer

**Bridging the gap between digital doubt and trust.**

Shopping Whisperer is an AI-powered trust analysis tool designed to protect online shoppers. By analyzing product URLs in real-time, the application detects "Red Flags" and "Green Lights" to provide users with a comprehensive Trust Score.

## 🚀 Features
- **Instant Trust Scoring:** Real-time analysis of product URLs using AI Agents.
- **Visual Signal Detection:** Clear categorization of Red Flags (risks) and Green Lights (positive signals).
- **Interactive UI:** A high-end Streamlit dashboard with modern design elements.
- **Document Analysis:** Optional upload feature for scam/truth knowledge base integration.

## 🛠️ Tech Stack
- **Language:** Python
- **Framework:** Streamlit (Web UI)
- **AI Engine:** Customized AI Agents (via `get_shopping_verdict`)
- **Deployment:** Streamlit Cloud / Hugging Face Spaces
